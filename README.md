🧊 FridgeVision
FridgeVision is an AI-powered smart refrigerator monitoring system that uses computer vision to identify food items, track freshness, and reduce food waste. It helps users manage their kitchen efficiently by providing real-time insights into what’s inside their fridge.

🚀 Features
📷 Food Detection – पहचानें fridge के अंदर मौजूद items using AI
⏳ Expiry Tracking – Track freshness and get alerts before food expires
🔔 Smart Notifications – Low stock और expiry alerts
📊 Inventory Management – Automatically maintain fridge inventory
🌐 User-Friendly Interface – Simple dashboard for easy monitoring

🛠️ Tech Stack
Frontend: React.js
Backend: Python (Flask / Django)
AI/ML: TensorFlow / OpenCV
Database: MongoDB / Firebase
Hardware (Optional): Camera module (Raspberry Pi)

⚙️ Installation
Clone the repository
git clone https://github.com/your-username/fridgevision.git
cd fridgevision
Install dependencies
pip install -r requirements.txt
Run the backend
python app.py
Run frontend
cd frontend
npm install
npm start

📸 How It Works
Camera captures image inside the fridge
AI model detects and classifies food items
System updates inventory database
Alerts are generated for expiry or low stock
